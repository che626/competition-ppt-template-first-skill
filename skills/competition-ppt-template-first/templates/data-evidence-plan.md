## Rules

- Use only verified source values. Preserve the unit, evaluation scope, dataset or test condition, and comparison baseline.
- Choose the visual form from the claim: comparison table for few categorical trade-offs, bar for discrete comparison, line for ordered change, scatter for a relationship, and a real screenshot when the evidence is interface behavior.
- Label the one conclusion the judge should notice. Do not make the audience infer the intended result from an unexplained chart.
- Keep model selection, performance, deployment, iteration, and system-test claims traceable through fact IDs or real evidence IDs.
- State a material limitation or caveat when the source requires one. Do not use decorative charts or invented values to fill space.
